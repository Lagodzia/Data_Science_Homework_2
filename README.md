# HSLU Fundamentals of Data Science: Graded Homework

Welcome to this repository. It provides you with all the files required to do the graded homework. Authored by Patricia Feubli.

For the homework, we use the Penguin data set contained in the palmerpenguins library. After installing and loading the palmerpenguins library, you will have access to the data set.

Please clone this repository in IntelliJ to get access to the required files.

Moreover, please watch this video below to get a short introduction to how to use the Rmarkdown: link

First of all, you need to decide on what you would like to do. Take a look at the data set and design a question you would like to answer with the data set. You are completely free in choosing a question. Decide on the approach you want to apply.

You will need to hand in two files: a Rmarkdown file and the corresponding html file. I created a template for the Rmarkdown file. The Rmarkdown documents your analysis, your thoughts on choices you made, as well as your evaluation and interpretation of the results.

The homework will be graded using the following criteria:
1. Question + performance metric stated.
2. Two labeled EDA (explorative data analysis) plots.
3. Brief missing value /quality note.
4. Simple, sensible cleaning.
5. Train/test split or k selection for clustering.
6. One reasonable model run without errors.
7. Correct performance metric.
8. 2–4 sentences for the interpretation of results/errors.
9. Rmd knits cleanly (no errors; seed set).
